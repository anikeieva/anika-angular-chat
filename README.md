# Anika Angular Chat
Online chat based on Angular, Soket.IO, Express, Mongoose (not finished yet)

## How to build server

```
cd server

#install:
npm install  
gulp build 
 
#run
npm start
```

## How to build client

```
cd client

#install:
npm install  
 
#run
npm start
```
